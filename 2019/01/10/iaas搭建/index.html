PAAS/docker平台

起始环境
crt 至少应该连接四台机器

1 ftp源即IAAS控制节点
2 server (vnc1)
3 client (vnc2)
4 regiest (vnc3)

进入平台后
ftp源搭建 创建/opt/docker目录 拷贝光盘下得所有软件到此目录

准备工作 需要在三台vnc上都执行

修改yum
rm -rfv *
vi yum.repo

[centos]
name=centos
baseurl=ftp://192.168.10.10/centos
gpgcheck=0
enabled=1
[docker]
name=docker
baseurl=ftp://192.168.10.10/docker/docker
gpgcheck=0
enabled=1

iptables -F
iptables -X
iptables -Z
/usr/sbin/iptables-save
修改系统内核
vi /etc/sysctl.conf
显示内核信息
sysctl -p
yum install docker-io -y
修改配置文件———————————–注意REGHOST_IP就是（VNC中regiest的ip）
vi /etc/sysconfig/docker
ADD_REGISTRY=’–add-registry REGHOST_IP:5000’
INSECURE_REGISTRY=’–insecure-registry REGHOST_IP:5000’
