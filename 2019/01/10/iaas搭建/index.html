iaas搭建

IAAS平台搭建

准备步骤和网卡要求

1.创建2台虚拟机分别叫做controller compute
controller 为控制节点
compute为计算节点

controller 和compute都需要2张网卡
一张网卡连接内网一张连接外网

contronller网卡1 192.168.100.10 网卡2 192.168.200.10

compute网卡2 192.168.100.20 网卡2 192.168.200.20

controller yum源配置

进入controller终端

cd /opt
mkdir iaas centos7  
rm -rf /etc/yum.repos.d/*            
vi　/etc/yum.repos.d/yum.repo

配置yum.repo

[centos]
name=centos
baseurl=file:///opt/centos
gpgcheck=0
enabled=1

[iaas]
name=iaas
baseurl=file:///opt/iaas-repo
gpgcheck=0
enabled=1

挂载光盘

mount -o loop /dev/sr0 /opt/centos
mount -o loop /dev/sr1 /opt/iaas

关闭防火墙

setenforce   0    
systemctl stop firewalld
systemctl disable firewalld

ftp配置

控制节点yum list 成功后 装ftp
装完ftp 后

vi  /etc/vsftp/vsftpd.conf

在最后一行添加

anon_root=/opt


service vsftpd restart 
chkconfig vsftpd on

openrc.sh配置

安装xiandian

yum install -y iaas-xiandian

进入openrc.sh

 vi /etc/xiandian/openrc.sh

##--------------------system config--------------------##
##Controller Server Manager IP. example:x.x.x.x
HOST_IP=192.168.100.10

##Controller Server hostname. example:controller
HOST_NAME=controller

##Compute Node Manager IP. example:x.x.x.x
HOST_IP_NODE=192.168.100.20

##Compute Node hostname. example:compute
HOST_NAME_NODE=compute

##--------------------Rabbit config ------------------##

RABBIT_USER=openstack

##--------------------Nova config----------------------##
##Password for Mysql nova user. exmaple:000000

##Password for Keystore nova user. exmaple:000000

##--------------------Neturon config-------------------##
##Password for Mysql neutron user. exmaple:000000

##metadata secret for neutron. exmaple:000000

以上是更改之后的配置文件

将次文件用scp命令发给compute节点

scp /etc/xiandian/o** root@compute:/etc/xiandian

controller节点服务的安装

然后进行安装脚本

iaas-pre-host.sh

然后重启

reboot

重启完成后安装mysql keystone glance nova neutron
安装mysql

iaas-install-mysql.sh

安装keystone

iaas-install-keystone.sh

安装完成后获取一下

source /etc/keystone/admin-openrc.sh

安装glance

iaas-install-glance.sh

安装nova

iaas-install-nova-controller.sh

安装neutron

iaas-install-neutron-controller.sh

安装neutron gre

iaas-install-neutron-controller-gre.sh

最后安装

iaas-install-dashboard.sh

到这里controller节点配置全部完成
接下来是compute节点配置

compute yum源配置

直接修改yum源为用ftp连接controller的yum源

rm -rf /etc/yum.repos.d/*        
vi　/etc/yum.repos.d/yum.repo

配置文件如下

[centos]
name=centos
baseurl=ftp://192.168.10.10/centos/
gpgcheck=0
enabled=1
[iaas]
name=iaas
baseurl=ftp://192.168.10.10/iaas-repo/
gpgcheck=0
enabled=1

compute 服务安装

yum list能列出后安装xiandian

yum install-y iaas-xiandian

安装iaas脚本

iaas-pre-host.sh

重启
root

之后安装nova 和neutron

iaas-install-nova-compute.sh
 iaas-install-neutron-compute.sh
 iaas-install-neutron-compute-gre.sh

完成之后

云平台搭建就完成了
保证自己主机p和云平台内网互通
在ie浏览器输入
192.168.100.10/dashboard
接下来就是页面操作了
